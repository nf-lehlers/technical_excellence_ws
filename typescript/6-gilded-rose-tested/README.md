# TDD-Starter (TypeScript)

Minimales Beispiel für testgetriebene Entwicklung: eine kleine Domäne (`Thing`) und **ein grüner Beispieltest** (`thing.test.ts`).

## Voraussetzungen

- **Node.js** (LTS)
- **npm**

## Stack

Tests mit **Vitest**, TypeScript.

## Einrichtung und Tests

```bash
npm install
npm test
```

Erwartung: ein bestandener Test.

## Watch-Modus (optional)

Tests bei Änderungen neu ausführen:

```bash
npm run test:watch
```
