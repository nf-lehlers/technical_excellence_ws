# TDD-Starter (Java)

Minimales Beispiel für testgetriebene Entwicklung: eine kleine Domäne (`Calculator`) und **ein grüner Beispieltest** (`CalculatorTest`).

## Voraussetzungen

- **JDK 21** (entspricht `maven.compiler.release` in `pom.xml`)
- **Apache Maven**

## Stack

Tests mit **JUnit 5**, Build mit **Maven**.

## Tests ausführen

Im Projektverzeichnis dieses Starters:

```bash
mvn test
```

Erwartung: Build erfolgreich, Tests grün.
